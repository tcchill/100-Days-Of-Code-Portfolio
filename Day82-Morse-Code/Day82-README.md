# Day 82 – Morse Code Converter 🔤

A command-line tool that converts text to Morse code and vice versa.

## Features
- Text → Morse code
- Morse code → Text
- Handles letters, numbers, and punctuation

## Tech Stack
- Python 3
- Dictionary-based encoding/decoding

## How to Run
```bash
python main.py
```

## What I Learned
- String manipulation in Python
- Building encode/decode logic from scratch
- Handling edge cases in user input
